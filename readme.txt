== Description ==

This widget that displays an archive for the current selected category. On non-category pages a standard archive is displayed. 

*   On a post page the widget will only display an archive for posts in that category
*	On a category listing the widget will only display an archive for posts in that category
*   On the front page or in a search result or any other page that doesn't reside in a category a standard archive will be displayed.

== Installation ==

1. Upload `subzane-categorized-archive-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar
